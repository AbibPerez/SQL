# 1.3 - Tipos de Datos SQL
En SQL, a cada columna se le debe asignar un tipo de dato que defina el tipo de datos que puede almacenar, como enteros, fechas, texto o valores binarios. Elegir el tipo de dato correcto es crucial para la integridad de los datos, el rendimiento de las consultas y la indexación eficiente.

## 1. Tipos de datos numéricos
Los tipos de datos numéricos son fundamentales para el diseño de bases de datos y se utilizan para almacenar números, ya sean enteros, decimales o de punto flotante. Estos tipos de datos permiten operaciones matemáticas como suma, resta, multiplicación y división, lo que los hace esenciales para gestionar datos financieros, científicos y analíticos.

### Tipo de dato numérico exacto
Se utilizan tipos numéricos exactos cuando se necesitan valores numéricos precisos, como para datos financieros, cantidades y conteos. Los siguientes son algunos comunes.

| Tipo de Datos | Descripción                           |
| ------------- | ------------------------------------- |
| `BIGINT`      | Valores enteros grandes               |
| `INT`         | Valores enteros estándar              |
| `SMALLINT`    | Valores enteros pequeños              |
| `TINYINT`     | Valores enteros muy pequeños          |
| `DECIMAL`     | Valores exactos                       |
| `NUMERIC`     | Valores precisos, similar a `DECIMAL` |

##### Ejemplo
```SQL
    CREATE TABLE ProductosOferta (
        ProductoID INT PRIMARY KEY,
        Cantidad SMALLINT,
        PrecioUnitario DECIMAL(10,2),
        MontoTotal DECIMAL(10,2)
    );
```

### Tipo de dato numérico exacto
Estos tipos se utilizan para almacenar valores aproximados, como mediciones científicas o grandes rangos de datos que no requieren precisión exacta.

| Tipo de Datos | Descripción                                |
| ------------- | ------------------------------------------ |
| `FLOAT`       | Valores numéricos aproximados              |
| `REAL`        | Similar a `FLOAT` pero con menos precisión |

##### Ejemplo
```SQL
    CREATE TABLE Mediciones (
        SensorID INT,
        Temperatura FLOAT,
        Humedad REAL
    );
```
## 2. Tipos de datos de caracteres y cadenas
Los tipos de datos de caracteres se utilizan para almacenar texto o datos basados en caracteres. La elección entre tipos de datos de longitud fija y variable depende de la naturaleza de tus datos.

| Tipo de Datos  | Descripción                                             |
| -------------- | ------------------------------------------------------- |
| `Char`         | Caracter de longitud fija máxima de 8000 caracteres     |
| `Varchar`      | Caracter de longitud variable máxima de 8000 caracteres |
| `Varchar(max)` | La longitud máxima de `2^31 - 1` caracteres             |
| `Text`         | La longitud máxima es de 2,127,483,647 caracteres       |

##### Ejemplo
```SQL
    CREATE TABLE InfoEmpleado (
        EmpleadoID INT PRIMARY KEY,
        PrimerNombre VARCHAR(50), 
        Apellido CHAR(30),
        BIO TEXT
    );
```
