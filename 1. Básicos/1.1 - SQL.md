# 1.1 - SQL
El Lenguaje de Consulta Estructurada (Structured Query Language - SQL) es el lenguaje estándar utilizado para interactuar con bases de datos relacionales.
+ Principalmente se usa para gestionar datos. Ya sea que quieras crear, eliminar, actualizar o leer datos, SQL proporciona comandos para realizar estas operaciones. 
+ Ampliamente soportado en diversos sistemas de bases de datos como __[MySQL](https://www.mysql.com/)__, __[Oracle](https://www.oracle.com/latam/database/)__, __[PostgreSQL](https://www.postgresql.org/)__, __[SQL Server](https://www.microsoft.com/en-us/sql-server/sql-server-downloads)__ y muchos otros.
+ Funciona principalmente con bases de datos relacionales (los datos se almacenan en forma de tablas).

## Escribir la primera consulta SQL
Antes de ejecutar consultas SQL necesitas configurar un servidor de bases de datos como MySQL, PostgreSQL o SQLite. Aquí vamos a usar el servidor MySQL. Sigue los siguientes pasos para configurar un entorno SQL básico:
+ [Instala MySQL en tu sistema](#)
+ [Inicia MySQL Server](#)
+ [Accede a la línea de comandos MySQL](#)

Una vez configurado tu entorno MySQL, puedes escribir tu programa SQL. A continuación se muestra el ejemplo para mostrar "Hola, mundo" usando SQL.<br>

1. Crea una base de datos
```SQL
    CREATE DATABASE test_db;
```
2. Utilizar la base de datos
```SQL
    USE test_db;
```
3. Crear una tabla
```SQL
    CREATE TABLE saludos (
        id INT AUTO_INCREMENT PRIMARY KEY,
        message VARCHAR(255)
    );
```
4. Inserta el mensaje 'Hola, mundo' en la tabla `saludos`
```SQL
    INSERT INTO saludos (mensaje)
    VALUES ('Hola, mundo');
```
5. Recupera el mensaje de la tabla `saludos`
```SQL
    SELECT mensaje FROM saludos;
```
Salida:
```TXT
    +---------------+
    | mensaje       |
    +---------------+
    | Hola, mundo   |
    +---------------+
```
